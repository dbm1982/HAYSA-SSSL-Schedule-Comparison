# HAYSA–SSSL Schedule Comparison System

This repository contains the automated pipeline that scrapes schedules from:

- **SSSL (South Shore Soccer League)**
- **HAYSA / TeamSideline**

…and produces a **daily mismatch report** showing any differences between the two systems.

The entire process runs automatically in **GitHub Actions** every morning and can also be run manually.
